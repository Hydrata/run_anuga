# Data Licensing

## Elevation Data

The example DEM (`inputs/dem.tif`) is a 200x200m clip of 1m resolution
elevation data in EPSG:28356 (GDA94 / MGA zone 56), sourced from the
[Geoscience Australia ELVIS portal](https://elevation.fsdf.org.au/).

**License:** [Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/)

**Attribution:** Commonwealth of Australia (Geoscience Australia) 2024.
Elevation data sourced from the ELVIS portal (elevation.fsdf.org.au).

## Synthetic Data

The boundary, inflow, and scenario configuration files in this example
are synthetic data created for demonstration purposes and carry no
additional licensing requirements beyond the repository's MIT license.
